# ScalR Workshop Website

This is the website for the 1st Workshop on Test-time Scaling and Reasoning Models (ScalR), to be held at the 2nd Conference on Language Modeling (COLM) 2025 in Montreal on October 10th, 2025.

The website is built using the TheEvent Bootstrap template and is hosted on GitHub Pages.

## Local Development

To run this website locally:
1. Clone the repository
2. Open index.html in your web browser
